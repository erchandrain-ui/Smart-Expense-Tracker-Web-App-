# 💰 Smart Expense Tracker Web App

A modern Flask-based expense tracking application that helps users manage daily expenses, analyze spending habits, generate reports, and predict future expenses using data analytics.

## 🚀 Features

* Add and manage expenses
* Categorize spending
* Dashboard with expense summary
* Category-wise breakdown
* Recent transaction logs
* Analytics and charts
* CSV import support
* Expense reports
* Future expense prediction
* Responsive Bootstrap UI

## 🛠️ Tech Stack

* Python
* Flask
* Pandas
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/erchandrain-ui/Smart-Expense-Tracker.git
cd Smart-Expense-Tracker
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
